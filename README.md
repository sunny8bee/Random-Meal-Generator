🔵프로젝트 소개


하루 중 음식을 고민하는 시간으로 얼마나 할애하곤 하시나요?
이때 뭐먹을 끼니🤔?가 등장해 여러분의 선택을 도와드릴 예정이랍니다!

🟢팀원 : 권승원, 남동하, 성유미, 양재연, 오유림, 정현지

🟡개발 동기


우유부단한 끼니 선택장애 극복을 위해 openAPI를 이용하여 프로젝트를 만들었습니다.

🟠느낀점


모두가 처음 만들어낸 프로젝트로 미숙함은 있었지만, 서로의 의견을 존중해주며 소통하여 프로젝트를 완성해 나갈수 있었습니다.

🟣우리만의 특별한 기능


인기검색어 기능 구현함으로써 인기 검색 키워드 제시
